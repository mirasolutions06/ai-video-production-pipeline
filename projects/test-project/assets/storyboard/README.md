# Storyboard Images

## How This Works

Storyboard images are the key to visual consistency across your video. They bridge
Gemini's image generation with Kling's video generation.

## Naming Convention

- `scene-1.png` — First frame fed into Kling to start the clip
- `scene-1-lastframe.png` — AUTO-GENERATED by pipeline from Kling output. Open in Gemini to generate scene 2.
- `scene-2.png` — Generated by Gemini looking at scene-1-lastframe.png
- Repeat for all scenes.

## Workflow

1. Add `../reference/style.jpg` (sets the visual vibe)
2. Switch to Gemini in Antigravity: "Generate scene 1 image in the style of [describe reference]"
3. Save Gemini's output here as `scene-1.png`
4. Run pipeline — it generates the Kling clip and saves `scene-1-lastframe.png`
5. Show Gemini the lastframe: "Continue this scene naturally, generate scene 2"
6. Repeat

## Image Requirements

- Format: PNG (preferred) or JPG
- Resolution: Minimum 768x768, ideally 1080x1920 for 9:16 or 1920x1080 for 16:9
- Content: Static, high-quality image that represents the start of the scene

## Auto-Discovery

The pipeline automatically finds all `scene-N.png` files in this folder in order.
You do not need to update config.json when adding new scenes — just drop the PNG here.
